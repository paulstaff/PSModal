PSModal.js
Easy, Beautiful Modal Windows
PSModal.js provides you with an easy-to-use modal window that looks good in every browser (well, maybe I haven't tested it in IE, but does anyone really use IE?). Generating a modal with PSModal involves calling one function; it's that simple.
Open Test Modal
Using PSModal.js is easy, just include the PSModal.js and PSModal.css files in your project, create title, content, and options (optional) variables to pass in to the function, and then call PSModal.open(title, content, options).
